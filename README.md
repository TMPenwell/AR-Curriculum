# Game Day Lab — An Augmented Reality Marketing Module

Game Day Lab turns a marketing classroom into a small creative agency. Over seven lessons, students take a brief from a fictional minor‑league sports client — the **Riverside Rapids** — and design, build, publish, and measure a branded augmented reality (AR) Lens for Snapchat. No coding and no prior AR experience required.

It is built for undergraduate marketing and advertising courses, structured around **Kolb's Experiential Learning Cycle** (*Do it → Reflect on it → Name it → Build it*), and teaches two no‑code build paths: **EasyLens** (browser, no install) and **Lens Studio** (desktop, more control).

## How to use it

1. **Start with the Teacher's Guide.** It explains the structure, the Kolb cycle, the two build paths, the fictional client, and includes facilitator notes and answer keys.
2. **Choose your path.** The Guide's **"Choosing Your Path"** section maps lessons to the time you have:
   - **One class session (~75 min):** Lesson 3 (build in EasyLens) + Lesson 5 (publish).
   - **A short unit (3–4 sessions):** Lessons 1, 2, 3, 5, 6 — the core *brief → brand & IP → build → publish → measure* arc.
   - **Add a marketing/career layer (~1 session):** the Lesson 6 organic‑vs‑paid extension + Snap Focus.
   - **The full unit (7 sessions):** Lessons 1–7, including the Lens Studio control path and the capstone pitch.
3. **Hand students the Workbook.** It's the fill‑in companion — briefs, design decisions, reflections, and end‑of‑lesson knowledge checks. (Answer keys and teaching notes live in the Teacher's Guide, not the Workbook.)
4. **Use the step‑by‑step tutorials.** Every build walkthrough (EasyLens, Lens Studio, and Publish) lives on a linked Scribe tutorial hub referenced inside both documents — share it, project it, or assign it on its own.

No Snapchat app or account is needed to begin; the early lessons run in any web browser.

> **Heads‑up:** Snapchat's tools (EasyLens, Lens Studio, and the publishing and analytics screens) change frequently. Confirm the current screens before class — the teaching steps stay the same even when on‑screen wording shifts.
